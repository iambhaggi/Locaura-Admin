# LOCORA – Hyperlocal Fashion Commerce Platform

LOCORA is a hyperlocal fashion e-commerce platform connecting
local clothing retailers with nearby customers.

## Platforms
- Customer Mobile App (React Native)
- Retailer Mobile App (React Native)
- Admin Web Panel (React.js)
- Backend API (Node.js + Express)
- Database (PostgreSQL + Redis)

## Folder Structure
/app      → React Native mobile apps
/backend  → Backend REST APIs
/admin    → Admin Web Dashboard

## Tech Stack
- React Native
- Node.js (Express)
- PostgreSQL
- Redis
- Firebase FCM
- Razorpay
- AWS S3 / Cloudinary
